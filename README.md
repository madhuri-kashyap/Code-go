# Real-Time Weather Checker using Python API
# The Real-Time Weather Checker is a python based application that fetches live weather data for any city using a openweather API. 
# This project demonstrates:
     - 1.API integration
     - 2.JSON data handling
     - 3.HTTP requests in python 
     - 4.Working with user input in terminal
# Users can check:
     - 1.Current temperature
     - 2.Weather condition 
     - 3.Humidity
# How it works: The application sends a request to the openweather API with the city name provided by the user. The API responds with the real-time weather data in JSON format, which is converted into dictonary format and displayed.
# STEPS:
- First visit openweather page and create an account, and generate an API key.
- Install required libraries in command prompt- pip install requests
- Go to python worksheet, write the code and replace with your own API key.
- Run the script
- In terminal section, enter the name of the city and get the result.
